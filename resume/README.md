## 源泉
个人简历。
## 简介
### 初始化项目
    npm install
### 运行编译
    npm start
访问：http://localhost:8080/
### 在线预览
[https://eternityspring.github.io](https://eternityspring.github.io)
### 打印预览
打开上述链接，在浏览器中按
    ctrl+p
